## All changes

### Company and Administration

| Added | Designated support users: Sage Intacct provides your company with 2 designated support users who can contact Support for technical questions and assistance.                                                                     No more passwords needed: Sage Intacct now supports passwordless authentication methods for signing into Intacct using Microsoft Entra.                                                                     Fiscal year rollover for document sequences: Document sequences now support fiscal year rollovers to help you manage your document sequences with year end transitions. |
| --- | --- |
| Updated | Permissions changes: We added new permissions for Accounts Receivable. Learn more about permission changes. |

### Accounts Payable

| Updated | Easy access to vendor payment files: You can now generate and access payment files for ACH and bank file payments in Accounts Payable.                                                                      Restricted users:  We've updated what actions restricted users can take in Accounts Payable.                                                                     Edit tax information in a draft bill: Create draft bills without entering tax solution information. You can update the draft later to enter a tax solution, tax schedules, and tax details.                                                                     Memo reclassification:  Learn how reclassifying the memo on a bill now affects the corresponding payment memo. |
| --- | --- |

### Accounts Receivable

| Added | Draft payments: You can now draft payments that you receive but are not ready to post. |
| --- | --- |
| Updated | Restricted users:  Learn about changes to what restricted users can do in Accounts Receivable. |

### Cash Management

| Added | Bank transaction assistant file import: Save time and increase control when you import bank transactions from a centralized location.                                                                     Canadian bank file payments: You can now pay vendors and employees electronically with bank file payments in CAD currency.                                                                     Match sequence report: Run a pre-built custom report on reconciliation matches. Learn about the report on reconciliation matches.                                                                      Import credit card transactions for reconciliation: Use the classic file import process to reconcile your credit card account—Early Adopter. |
| --- | --- |
| Updated | Cash Management transaction batches: Get a more granular view into summary journal posting. Read about batches created by transaction type. |

### Consolidation

| Added | Separate opening balance entries: Sage Intacct now separates opening balance entries from current period activity when consolidating after the first period of business activity. |
| --- | --- |

### Fixed Assets Management

| Added | Announcing Fixed Assets Management:  Automate and simplify the management  of your assets from acquisition to disposal with this new application.                                                                     Create assets from purchasing transactions: Streamline your data entry process by automatically creating assets from purchasing transactions such as vendor invoices. |
| --- | --- |
| Updated | Improved workflow for non-depreciable assets: We've simplified how non-depreciable assets are displayed and added a new setting for their disposal.                                                                     Other updates in Fixed Assets Management: Other updates include improvements to default list views, changes to the daily depreciation calculations, and more. |

### Help and Training

Audit Trail checkbox: To simplify your user experience, we removed a checkbox from the edit object field page for system-generated fields.  		                                                                     Smart event log changes: We improved how Smart events are logged to streamline your workflow by eliminating the requirement to install a separate package. |

### Taxes

| Added | Cash basis VAT capture for France: Capture taxes for your VAT transactions at point of payment or invoice with our tax capture enhancements for France.                                                                     Support for taxes in France: Capture and calculate domestic VAT in France with the new TVA Française standard tax solution. Companies using this tax solution can generate and export reports in the Accounting entries file (FEC) format to comply with tax reporting regulations.                                                                     Partial exemption for United Kingdom and France: Updated functionality lets you set percentages for partial exemption at the tax detail level, giving you increased flexibility to calculate your VAT taxes. |
| --- | --- |
| Updated | Taxes updates and enhancements: Update United Kingdom and France tax solutions with new changes, toggle offset tax credit option for South Africa, and edit tax information in draft billswith these new enhancements for Taxes. |
