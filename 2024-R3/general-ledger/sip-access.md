# Easy access to Sage Intacct Planning

[Basic budgeting vs. Sage Intacct Planning](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Basic_vs_budgeting_and_planning)

[Integrate with Sage Intacct](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Configure_SIBP)

General Ledger

Sage Intacct Planning is an add-on application that offers a collaborative environment for preparing and managing budgets.

If you have not subscribed to Sage Intacct Planning, talk to your Sage Intacct account manager for information about [subscribing](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Contact_your_account_manager).

View a [comparison of the features in Planning and the basic budgeting features in Intacct](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Basic_vs_budgeting_and_planning).

You can now access the Sage Intacct Planning log-in page directly from the Budget planning page.

###### ![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/2024/2024-R3-images/2024-R3-sip-access.png)

## How it works

To log in to Sage Intacct Planning:

1.  Go to **Budgets** \> **Budget planning**.
2.  Select **Open Sage Intacct Planning**.

The Sage Intacct Planning log-in page opens in another tab.

If you have permission to view the Budget planning page, you'll see the Planning link. You'll need a separate Sage Intacct Planning account to sign into the application.
