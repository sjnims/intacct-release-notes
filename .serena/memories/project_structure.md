## Key Files
- `extract.mjs` — All extraction logic (606 lines). Contains RELEASES config, URL helpers, DOM cleanup, Turndown setup, discover/extract phases, and CLI.
- `manifest.json` (per release) — Maps source URLs to output filenames, generated by discover phase.
- Each `.md` file — Extracted content with YAML frontmatter (`source`, `release`, `extracted`, `title`).
