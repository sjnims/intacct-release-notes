## Key Files

- `extract.mjs` — Scraping and extraction logic. Contains RELEASES config, URL helpers, DOM cleanup, Turndown setup, discover/extract phases, and CLI.
- `summarize.mjs` — AI summarization logic. Contains duplicated RELEASES config, Anthropic SDK calls, system prompts, and CLI. Requires `ANTHROPIC_API_KEY` env var.
- `manifest.json` (per release) — Maps source URLs to output filenames, generated by discover phase.
- Each extracted `.md` file — Content with YAML frontmatter (`source`, `release`, `extracted`, `title`).
- Each summary `.md` file — AI-generated content with frontmatter (`source: generated`, `release`/`year`, `generated`, `generator`, `type`).
