# User interface changes

In this release, we updated aspects of the user interface throughout Intacct for clarity and consistency.

| Product area | Change description | Old UI | New UI |
| --- | --- | --- | --- |
| Consoles | We changed the format of the following fields on the Manage instances list:                                                                                                                                                                                                               Flat                                                                                                                                                                                                                   Hierarchical | Button | Checkbox |
| Subscriptions | We renamed the subscription for electronic vendor payment services. | Outbound Payment Services | Vendor Payment Services |
| Accounts Payable                                                                 Purchasing | We renamed a column for accuracy in the Recurring transaction status report. | Invoice/doc date | Transaction date |
| Accounts Payable                                                                 Accounts Receivable                                                                 Order Entry                                                                 Purchasing | We renamed an option for the Transaction type filter in the following reports:                                                                                                                                                                                                               Recurring transaction status report (AP, AR, OE, PO)                                                                                                                                                                                                                   Recurring transaction forecast (AR, OE) | All bank accounts | All transaction types |
| Supplies Inventory | With expanding functionality in the Supplies workbench, we've modified some column headers. | Quantity                                                                                                                                                                                                                   Price                                                                                                                                                                                                                   Employee | Qty requested                                                                                                                                                                                                                   Unit price                                                                                                                                                                                                                   Requested by |
| Fixed Assets Management | We renamed a field used in the AP and Purchasing integrations to better fit its new options. | Asset creation | Asset action |
| Projects                                                                 WIP Management                                                                 With the Construction subscription | We moved a page in the Projects menu from the All tab to the Setup tab, and we renamed the page. | Project settings for WIP schedules | Include projects in WIP schedules |
| We updated some labels to make them clearer. | Update CFO values with PM values                                                                 Update CFO values with CFO values from prior period                                                                 Clear CFO values                                                                 Update PM values with project values                                                                 Update PM values with PM values from prior period                                                                 Clear PM values | Update CFO forecasts with PM forecasts                                                                 Update CFO forecasts with CFO forecasts from prior period                                                                 Clear CFO forecasts                                                                 Update PM forecasts with project values                                                                 Update PM forecasts with PM forecasts from prior period                                                                 Clear PM forecasts |
