# Purchase order document template now includes primary document summary and details

[Printed document templates](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=TOC_printed_document_templates)

construction

Improve clarity, visibility, and include key information from the primary document in generated Purchase order printed documents. View essential background information about the Purchase order to help streamline your printed documents that are sent internally or externally.

## What's changed

The following information is now available to be included in Purchase Order templates:

-   Primary document summary gives you a concise overview of the associated primary document.
    
-   Detailed breakdown of relevant details like document type, reference number, and key dates for improved traceability.
