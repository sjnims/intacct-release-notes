# New AR Customer Refund report object—CRW

[Accounts Receivable objects](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=reporting_object_glossary_ar)

ADVANCED REPORTING

You can now build custom reports on Accounts Receivable customer refunds using a new object available in the Custom Report Writer (CRW).

To get started, select **Customer Refund** as your primary data source when creating a report.

For details about available objects for Accounts Receivable custom reports, see the [object glossary](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=reporting_object_glossary_TOC).

## Permissions and other requirements

| Subscription | Customization Services or Platform Services                                                                 Accounts Receivable |
| --- | --- |
| Regional availability | All regions |
| User type | Business |
| Permissions | Custom Reports: List, View, Add, Edit, Delete |
