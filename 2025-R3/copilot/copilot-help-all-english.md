# Search help with Copilot—Expanded English language support

Exciting news! Search help with Copilot is now available for all supported English-speaking regions.

## Key benefits

Search help with Copilot is designed to help you find information within the Sage Intacct Help Center and offers the following benefits:

-   **Natural language understanding**: Copilot understands the nuances of Sage Intacct terminology and can interpret complex queries.
-   **Increased efficiency**: Significantly reduces search time and the need for multiple searches by delivering more relevant results.
    
-   **Intelligent recommendations**: Provides suggestions for related topics and articles to expand your knowledge.

## How it works

There's a new **Search help** option in the **Help & Support** menu.

![Shows the Help and Support menu, with the Search help menu option called out.](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/2024/2024-R4-images/2024-R4-copilot-search-help.png)

Selecting **Search help** opens the Copilot panel on the right side of your Intacct page.

![Shows an example of a help query and AI-generated response.](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/2024/2024-R4-images/2024-R4-copilot-search-panel.png)

Enter a query using natural language in the search box located at the bottom of the **Search help** panel. The more specific you are, the more accurate the results will be.

Copilot responds with an answer and provides links to additional information.

## Requirements

| Subscription | Search help with Copilot is accessible from any page within Sage Intacct. |
| --- | --- |
