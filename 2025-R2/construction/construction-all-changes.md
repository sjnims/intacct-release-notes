# 2025 R2 updates for Sage Construction

A comprehensive list of all construction-related updates for this release.

| Sage Intacct | Feature |
| --- | --- |
| Construction | Vendor compliance                                                                                                                                                                                                                   Enhanced support for smart events and smart rules on  compliance records: You can now set up smart events and smart rules to streamline your compliance processes and ensure that your compliance records are accurate and up to date.                                                                     Generate lien waivers for secondary vendors: You can now generate lien waivers for secondary vendors on AP bills, all payments, or joint checks. |
| New Primary document details tab on primary documents in Purchasing :  You can now access all purchasing change order and invoice details for primary documents in one central location. |
| Stay on project change order after posting:  When you post or draft a project change order, you can select Post & continue (or Draft & continue) to keep your document open and continue working. |
| Import project change requests: Save time and upload multiple change orders at once. |
| Project manager permissions for project contract tables: Project managers get Add, Edit, View, List, and Delete permissions. |
| Projects |
| Time and Expenses split into two applications: Previously, Time and Expenses were combined into a single application. The subscription still includes both applications but you access them using dedicated menus.​ |
| Introducing Automated WIP relief: Automatically reclassify WIP balances upon project completion  to close the books faster |
| Faster contract setup when duplicating project contracts: When you duplicate a project contract, Intacct can now copy mapped tasks for time and material contract lines. |
| New time and material summary totals workflow: This new workflow enables you to manually refresh summary totals and increases the number of expenses in each transaction type to 4000. |
| WIP management | See historical data for projects in work in progress (WIP) management: You can now view project history for WIP projects, and you can export and import WIP schedule information more easily. |
| New and enhanced ICRW Report library reports: Now available in the ICRW Report library, new Fixed Assets Management and enhanced  Construction Work In Progress (WIP) reports. |
| Sage for Construction | Features |
| Sage Construction Management | Enhancements in Sage Construction Management: This release includes many new features and enhancements. |
| Sage Field Operations | Improved work order activity view in Sage Field Operations: See more data and in-context details for work orders. |
| Sage Paperless for Construction | Enhancements in Sage Intacct Paperless: Use Advanced Tax configurations, and import credit cards to create invoices in the APFlow module. |
| Sage Construction Payroll | Enhancements in Sage Intacct Construction Payroll: Learn about the latest changes in Construction Payroll. |
| Sage BidMatrix | Help and release notes |
| Sage Estimating | Release notes and documentation |
| Sage 100 Contractor | Release notes and documentation |
| Sage 300 Construction and Real Estate | Release notes and documentation |
