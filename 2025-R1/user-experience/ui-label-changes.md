# User interface changes

In this release, we updated aspects of the user interface throughout Intacct for clarity and consistency.

| Product area | Change description | Old UI | New UI |
| --- | --- | --- | --- |
| Accounts Payable, Accounts Receivable, Cash Management, Order Entry, and Purchasing | For multi-currency companies, the Posting details tab now includes the base currency and debit and credit amounts in the base currency. | The Posting details tab only showed the transaction currency and debit and credit amounts in the transaction currency. | Base currency                                                                                                                                                                                                                   Debit base amount                                                                                                                                                                                                                   Credit base amount |
| Company | We added a checkbox on the Entity information page that will be used for a future reporting feature. |  | Limited entity |
| Consoles | On the Instance Setup Wizard, we moved the Contact email field to a different tab. | Contact tab | Login tab |
| On the Instance Setup Wizard, we made an individual field label into 2 field labels. | Contact name | Contact first name                                                                                                                                                                                                                   Contact last name |
| Fixed Assets Management | On the Fixed Assets Management Configuration page, we renamed a section title and field label to include new functionality. | Disposal settings | Asset transfer and disposal settings |
| Journal for non-depreciable assets | Journal for transfers and non-depreciable assets |
| Inventory Control | On the Configure Inventory page, we updated a label to better reflect its functionality. | MIV costing updates | Maintain inventory valuation (MIV)Â costing updates |
| On the Inventory Utilities page, we renamed a section on the Transaction definitions tab. | Transaction flow graphs | Transaction workflows |
| On the Inventory Utilities page, the generated report output file was renamed. | TransactionGraph.csv | TransactionWorkflows.csv |
| Projects | On the Project information page, we renamed a tab. | Resource and pricing tab | Resource and billing tab |
| On the Project information page, we moved and combined sections from one tab to a different tab. | Invoices options tab | Resource and billing tab |
| WIP Management | In WIP schedules, we renamed a field to align with industry terminology. | Estimated cost at completion | Total revised estimate |
| In WIP schedules, we updated the names of 2 of the buttons above the table. | Update CFO values with CFO prior WIP values                                                                                                                                                                                                                   Update PM values with PM prior WIP values | Update CFO values with CFO values from prior period                                                                                                                                                                                                                   Update PM values with PM values from prior period |
