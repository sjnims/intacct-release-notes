# Enhancements in Sage Construction Management

[Integration tips for Sage Construction Management](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Tips_Integrate_SageConstrMgt)

CONSTRUCTION

Sage Construction Management is an all-in-one cloud solution that provides a complete set of project management features. This solution aligns your operations and finance data so you can win more business and execute projects more efficiently. For more information, visit the [Sage Construction Management web page](https://www.sage.com/en-us/products/sage-construction-management/).

## Details

In this release we've made enhancements to Sage Construction Management in the following areas.

### Browser

- The Project Home and Lead Home pages now include a link you can use to view the user access list and modify user access.
- A new section in **Settings** > **Developer Mode** shows API usage for the last 30 days.
- External team members can now add notes and linked files to their project to do details and update the status of the project.
- The file size limit for imports has been increased to 500 MB, so you can import larger drawings and other files.
- New location fields are available for importing estimates.
- A new option lets you remove labor timecards from project timesheets and import them through the Accounting Link for Sage Intacct.
- A new **Show in Portal** property is included in **Photos** > **Album View** > **Show** next to the Album Name for Project photos.

### Mobile app

- You can now edit, rotate, and crop photos uploaded through the mobile app.
- A new option lets you remove labor timecards from project timesheets and import them only through the Accounting Link.

### TeamLink portal

### Accounting Link for Sage Intacct

- You can now create contract items using Sage Construction Management cost code division or major.
- You can now add SIC cost codes to Sage Construction Management cost codes when importing ERP job to date costs.
- New posting preferences when posting AR transactions.
- In equipment timecards:
  - See the equipment type in the equipment list.
  - Filter by owned/rental with cost rate properties.
  - Add more information for rental equipment: Default Item, GL Account, and Cost Type.
- The truncation setting is now applied when creating project contract lines in Sage Intacct.
- Company Code and Employee ID can be now updated when manually linking customers, vendors, and employees.
- Bill rate values that have been imported from Sage Intacct are revised in Sage Construction Management through Time Sheets maintaining the values that have been added in Sage Construction Management.
- In the Mobile App, the Add options in labor timecards are hidden if Sage Intacct Project Timesheets are imported through the Accounting Link.

### Accounting Link for Sage 300 Construction and Real Estate

- You can now set labor hour budgets in the Accounting Link for Sage 300 Construction and Real Estate when creating estimates.
- You can now export labor timecards from Sage Construction Management to a file and then import them to Sage 300 Construction and Real Estate.

### Accounting Link for Sage 100 Contractor

You can now set labor hour budgets in the Accounting Link for Sage 100 Contractor when creating estimates.

### Analytics

- New detailed analytic report: **Revised Contract and Projected Cost Overview Without Tax Using Bill Rates**.
- The **Revised Contract and Project Cost Overview** and the **Revised Contract and Project Cost Overview Without Tax** reports have been moved to the Overview tab.

To learn more about these and other enhancements, see the [Sage Construction Management release notes](https://help.sagecm.corecon.com/Content/ReleaseNotes/RNs-home.htm).
