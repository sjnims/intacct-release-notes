# Consolidated dashboards in Sage Intacct Project Intelligence

For multi-entity organizations operating in different base currencies, new dashboards are available that display consolidated data across all entities in a single base currency. The main dashboards provide data for individual locations in their functional currencies, while the new consolidated dashboards present a unified view of data from all entities in a common currency.

[![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/2025/2025-R1-images/2025-R1-project-intelligence_thumb_0_100.png)](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/2025/2025-R1-images/2025-R1-project-intelligence.png)

## Details

The main dashboards update automatically when you post. Consolidated dashboards display data from completed consolidations, and are updated only after running the consolidation process.

If you make changes for a previously consolidated month, you need to run the consolidation process again to update the dashboards.
