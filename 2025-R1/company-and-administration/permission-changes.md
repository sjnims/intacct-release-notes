# Permission changes

[Assign user permissions for new features (4:05)](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/Company_and_Administration/2025-R1-permission-changes.htm#)

Company and Administration

You can find information about the permissions needed for each new feature in the Permissions and other requirements section of its associated release note.

## Updated

We've updated aspects of the following permissions.

### Projects and Time and Expenses

These permissions are only for customers that have the embedded version of Sage Intelligent Time enabled.

-   **My timesheets:** List, View, Add, Edit, Delete
    
    Previously, these permissions affected the My timesheets feature only. Now, these permissions affect the My timesheets and My time clocks features.
    
-   **Staff timesheets:** List, View, Add, Edit, Delete
    
    Previously, these permissions affected the Staff timesheets feature only. Now, these permissions affect the Staff timesheets, Staff time clocks, and Rate cards features.
    

We made this change for the My time clocks and Staff time clocks features during 2024 Release 4.

## Added

All new permissions are automatically off by default, unless otherwise noted.

### Accounts Payable

-   **Deferred adjustments:** List, View, Add, Edit, Delete
    
    Allows you to enable [Deferred adjustments](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/Accounts_Payable/2025-R1-ap-prepaid-france.htm) and create Deferred adjustments templates.  
    

-   **Reports, Deferred adjustments forecast report:** Run
    
    Allows you to run [Deferred adjustments](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/Accounts_Payable/2025-R1-ap-prepaid-france.htm) forecast reports.
    

### Accounts Receivable

-   **Deferred adjustments:** List, View, Add, Edit, Delete
    
    Allows you to enable [Deferred adjustments](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/Accounts_Receivable/2025-R1-ar-prepaid-france.htm) and create Deferred adjustments templates
    

-   **Reports, Deferred adjustments forecast report:** Run
    
    Allows you to run [Deferred adjustments](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/Accounts_Receivable/2025-R1-ar-prepaid-france.htm) forecast reports.
    

### General Ledger

-   **Variance analysis:** Run
    
    Allows you to run [Sage Copilot Variance Analysis](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/Copilot/2024-R4-copilot-variance-analysis.htm).
    
-   **GL regularization account reclassification report:** Run
    
    Allows you to run the [GL regularization account reclassification report](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/General_Ledger/2025-R1-gl-dynamic-accounts.htm).
    

### Order Entry

-   **Revenue transaction entries:** List, Edit, Post, Delete  
    
    Allows you to [recognize external revenue](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/RevenueManagement/2025-R1-revenue-queue.htm).
    
-   **Bulk action runs**: List, View  
    
    Enables you to view bulk actions associated with [recognizing external revenue](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_1/RevenueManagement/2025-R1-revenue-queue.htm).
