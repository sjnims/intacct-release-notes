# New in Interactive Visual Explorer (IVE)

[Welcome to Interactive Visual Explorer](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=IVE_overview)

[IVE learning journey](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=IVE-learning-journey)

[IVE visualization types](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=IVE_visual_type)

ADVANCED REPORTING

IVE requires a separate subscription. To acquire or enable an IVE subscription, [contact your Sage Intacct account manager](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Contact_your_account_manager).

## Key benefits

- **Conditional formatting and blending** allow you to highlight important events and combine formatting rules.
- **Emojis and icons** allow you to highlight data points in Table, Pivot, or Tile visualizations.
- **New and enhanced filtering** allows you to Exclude filter values, Filter NULL values, and apply Top Bottom N filters.
- **Parameters** allow you to hold and manage a current value or values for use in multiple places on a canvas. When you bind a parameter to a filter, you can pass a selected value from one canvas to another.

## What's changed

There are newly updated visualization type icons.

## How it works

1. Follow the instructions in [Create visualizations in IVE](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=IVE_new_visualizations).
2. Perform the following tasks:
   - [Add conditional formatting and blending in IVE](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=IVE-conditional-formatting).
   - [Add emojis and icons to IVE visualizations](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=IVE-add-emojis-icons).
   - [Apply filters in IVE](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=IVE_filter_apply).
   - [Create and bind parameters to filters in IVE](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=IVE-create-bind-parameters).
