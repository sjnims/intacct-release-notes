# Create custom reports on payment approval history

[Accounts Payable and Payments objects](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=reporting_object_glossary_payment)

Accounts Payable

You can now build custom reports on Accounts Payable payment approvals using a new object in the Custom Report Writer (CRW).

To get started, select **AP Payment Approval History** as your primary data source when creating a report.

For details about available objects for Accounts Payable custom reports, see the [object glossary](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=reporting_object_glossary_TOC).

## Permissions and other requirements

| Subscription          | Customization Services or Platform Services Accounts Payable |
| --------------------- | ------------------------------------------------------------ |
| Regional availability | All regions                                                  |
| User type             | Business                                                     |
| Permissions           | Custom Reports: List, View, Add, Edit, Delete                |
