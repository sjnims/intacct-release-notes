# Enhancements in Sage Intacct Data Flow

- The **HubSpot to Intacct Order Entry Wizard** now supports multiple lines with different products.
- The **HubSpot to Intacct Customers w Contacts Wizard** now supports updates to contact mapping.
- When mapping data from an external data source to Sage Intacct, the Data Table tool now appends new records to the data table. Existing data is not removed.

For more information, talk to your [Sage Intacct account manager](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Contact_your_account_manager).
