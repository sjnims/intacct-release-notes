# Sage Intacct Inventory Automation and Supplies Inventory

[Sage Intacct Inventory Automation](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=inventory-automation)

INVENTORY CONTROL

Integrating Sage Intacct Inventory Automation with Supplies Inventory helps you act on employee supply requests faster and more accurately by using mobile devices to scan barcodes on requested items.

## Details

Use the Supplies Intacct Inventory Automation mobile app to perform the following tasks for a requisition:

- Receive purchase orders when items are ordered to fulfill a request.
- Perform warehouse transfers when sending supplies that were in stock in another warehouse.
- Issue out supplies as they're consumed.

For more information about Sage Intacct Inventory Automation and Supplies Inventory, talk to your [Sage Intacct account manager](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Contact_your_account_manager) or visit the [Sage Intacct Inventory Automation web page](https://www.sage.com/en-us/sage-business-cloud/intacct/resources/datasheets/streamline-tracking-inventory-with-inventory-automation/).
