# Import project change orders with new template

Handle large volumes of data and easily upload and import multiple project change orders at once with a new import template.

## Details

You do not need to provide a project change order ID if you're using document sequences in Projects. If document sequences are not enabled, an ID is required.

## How it works

You can download a template and import project change orders from the Company setup checklist or from the Project change orders list.

### Download a template and import from the Project change orders list

1. Go to **Projects** \> **All** \> **Change management** \> **Project change orders**.
2. Select **Import**.
3. Select **Download template**.
4. Enter project change order information.
5. Back on the Project change orders import dialog, select **Import**.

### Download a template and import from the Company setup checklist

1. Go to **Company** \> **Setup** \> **Import data**.
2. Find the section Import projects, tasks, and timesheets.
3. Find Project change orders.
4. Select **Template**.
5. Enter project change order information.
6. Back on the Company setup checklist, select **Import**.
