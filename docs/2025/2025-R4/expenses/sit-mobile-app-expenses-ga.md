# Sage Intelligent Time and Expense mobile app supports employee expense automation—General availability

[Automate employee expenses](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Automate_employee_expenses)

[Sage Intelligent Time and Expense Mobile FAQs](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Mobile_FAQ_page)

[My expense reports](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Aa_TOC_my_expense_report)

[Approve expense reports](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=TOC_approve_expenses)

expenses

With automation, you can capture receipts, submit expense reports, and approve or decline them from a mobile device quickly. This enhancement aims to boost efficiency, reduce errors, and quicken reimbursements for employee expense reports.

## Key benefits

- Improved accuracy
- Faster reimbursements
- Enhanced compliance and visibility

## How it works

[Step 1. Download the app](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_4/Expenses/2025-R4-SIT-mobile-app-expenses-GA.htm#)

**Google app**

Get the Android app at this link: [https://play.google.com/store/apps/details?id=com.sage.intelligent.time](https://play.google.com/store/apps/details?id=com.sage.intelligent.time)

Or, search the Google Play Store for Sage Intelligent Time & Exp.

**Apple app**

Get the iOS app at this link: [https://apps.apple.com/us/app/sage-intelligent-time/id1523625682](https://apps.apple.com/us/app/sage-intelligent-time/id1523625682)

Or, search the Apple App Store for Sage Intelligent Time & Exp.

[Step 2. Log in to the app](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_4/Expenses/2025-R4-SIT-mobile-app-expenses-GA.htm#)

1. Open the **Sage Intelligent Time and Expense** mobile app.
2. Select **Log in with Intacct**.
3. Enter your **Company ID**.
4. Enter your **User ID**.
5. Enter your **Password**.
6. Select **Log in**.

[Step 3. Create an expense report](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_4/Expenses/2025-R4-SIT-mobile-app-expenses-GA.htm#)

1. Select **Create expense report**.
2. Enter information in the required fields.
3. Select **Save**.

[Step 4. Submit your expense report](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_4/Expenses/2025-R4-SIT-mobile-app-expenses-GA.htm#)

1. Select the side bar.
2. Select **Expense reports**.
3. Find your expense report and fully swipe to the right.

[Step 5. An admin approves or declines the expense report](https://www.intacct.com/ia/docs/en_US/releasenotes/2025/2025_Release_4/Expenses/2025-R4-SIT-mobile-app-expenses-GA.htm#)

1. Select the side bar.
2. Select **Expense approvals**.
3. Select an expense report.
4. Select **Approve** or **Decline**.
5. Optionally, enter a **Note**.

## Good to know

- You will have the same user type, role, and permissions in the Sage Intelligent Time and Expense mobile app as you do in Sage Intacct.
- Admins cannot manage employee expense reports using the Sage Intelligent Time and Expense mobile app. You can manage employee expense reports using Sage Intacct.

  For more information, go to [Staff expense reports.](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Aa_TOC_staff_expense_reports)

## Action required

To use expense reports in the Sage Intelligent Time and Expense mobile app, ensure that you enable electronic receipt inbounding. For more information, go to [Automate employee expenses.](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Automate_employee_expenses)
