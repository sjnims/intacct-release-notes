# The hidden gems of 2024

2024 was a year of exciting updates! Approximately 240 new features were created across all areas of Sage Intacct. These features have empowered you to streamline workflows, gain deeper insights, and unlock new possibilities for your business.

## Many of these ideas came from you

**You** were instrumental in shaping the innovations that came out last year! This list spotlights some hidden gems that were inspired by your valuable suggestions.

![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/Year-end/spotlight-catch.png)

### Limit contacts available in transaction dropdowns

Streamline processing Accounts Payable and Accounts Receivable transactions by limiting selectable contacts in dropdowns. To help prevent data entry errors, you can restrict contact options to only those associated with a vendor record, or the specific vendor selected for the transaction.

[Streamline your work with limited contacts](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_1/Accounts_Payable/2024-R1-ap-restrict-contacts.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-limit-contacts)

![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/Year-end/spotlight-calculator.png)

### Edit tax information in draft bills

Companies using Taxes can now edit tax information for bills in a draft state. Users can now save progress and refine tax details before posting the bill. Intacct ensures accuracy by validating tax information only upon bill posting.

[Learn how to edit tax information for draft bills](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_2/Accounts_Payable/2024-R2-tax-validation-relaxed-ap-bills.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-USA-tax-info-draft-bills)

![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/Year-end/spotlight-fishing.png)

### Unapply credits in Posted payments

You can unapply credits that were applied without an associated payment. Users can now quickly unapply credits of various types, review applied credits with greater detail, and easily navigate the unapplying process.

[Try out the unapply credits process](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_3/Accounts_Payable/2024-R3-unapply-credits-in-ap.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-unapply-credits)

![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/Year-end/spotlight-screen-desk.png)

### Void a payment directly from the Posted payments page

Voiding posted payments has gotten easier with a centralized location on the Posted payments page. You can now conveniently void payments directly from where they are typically viewed, improving overall efficiency.

[Void payments from the Posted payments page](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_3/Accounts_Payable/2024-R3-void-payment-from-posted-payments.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-void-payment-on-page)

![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/Year-end/connect-bank-rounded.png)

### Bank transaction assistant file import

The Bank transaction assistant file import offers a centralized space to import transactions, supports more file types, and gives you better control over imports with enhanced permissions and an activity log. The new import method offers improved efficiency and a more streamlined reconciliation workflow.

[Import files with the Bank transaction assistant](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_2/Cash_Management/2024-R2-file-import-v1.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-bank-transaction-assistant)

![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/Year-end/receipts-rounded.png)

### Usage insights

Usage insights gives you real-time data and valuable insights into your company's operations. You can analyze transaction volume, usage patterns, and user activity to optimize processes and identify growth opportunities.

[View usage insights for your company](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_3/Company_and_Administration/2024-R3-usage-insights.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-usage-insights)

### Quickly delete unneeded reporting periods

Instead of deleting periods individually, you can now select multiple reporting periods and delete them all with a single click. This significantly improves efficiency and saves time, especially when dealing with numerous reporting periods.

[Learn how to delete multiple reporting periods](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_3/General_Ledger/2024-R3-bulk-delete-reporting-pds.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-delete-reporting-periods)

![](https://www.intacct.com/ia/docs/en_US/releasenotes/Resources/Images/Year-end/spotlight-fixing.png)

### Trace original transaction documents

We've made it easier to track how purchase document workflows are processed by showing which document started each transaction. Better tracking helps with inventory reconciliation, which leads to better overall inventory management.

[Start tracking purchase documents through their workflows](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_3/Purchasing/2024-R3-PO-originating-document.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-track-PO-from-transactions)

### Version history for financial reports

Version history for financial reports allows you to track changes made to report settings over time. By comparing current and previous versions, users can easily identify and troubleshoot unexpected results, revert to previous working versions, and maintain the integrity of their financial reporting.

[Review the version history of financial reports](https://www.intacct.com/ia/docs/en_US/releasenotes/2024/2024_Release_1/Reporting-Financial/2024-R1-FRW-version-history.htm?utm_source=2024-hiddengems&utm_campaign=2024-hiddengems-TBD)
