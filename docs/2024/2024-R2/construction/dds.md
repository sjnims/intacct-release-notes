# Construction now supports DDS

Construction now supports the following Data Delivery Service (DDS) objects:

- AR retainage release
- AR retainage release entry
- AP retainage release
- AP retainage release entry
- Change request
- Change request entry
- Cost type
- Project change order
- Project contract
- Project contract line
- Project contract line entry
- Project contract rate table
- Project estimate
- Project estimate entry

## Requirements

| Subscription          | Construction Projects                              |
| --------------------- | -------------------------------------------------- |
| Regional availability | Australia Canada United Kingdom (EA) United States |
| User type             | Business user                                      |
| Permissions           | Application subscriptions: List, View, Configure   |
