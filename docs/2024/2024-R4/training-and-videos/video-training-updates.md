# New training and videos

At Sage Intacct, we get how vital it is to have in-depth and best practice learning that enhance your use of Intacct. To that end we recently:

- Provided access to new self-paced, on demand courses that guide you through specific tasks and features.
- Added new product demo videos to help you get up and running on a variety of tasks.

## New and updated training courses

You can find these new and updated training courses at [Sage University](https://sageu.com/SageIntacct/).

You'll need to [log into Sage University with your Sage account](https://www.intacct.com/ia/docs/en_US/help_action/Default.htm#cshid=Connect_Sage_University).

| Application                                                           | Course                                                                                                                             | Description                                                                                                                                                                                                    |
| --------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Consolidation                                                         | Creating, Running, and Reporting on Consolidations                                                                                 | Teaches you how to create consolidation books, run consolidations, and report on consolidation books.                                                                                                          |
| Creating, Running, and Reporting on Advanced Ownership Consolidations | Teaches you how to create ownership structures, modify consolidation books, run consolidations, and report on consolidation books. |
| Projects                                                              | Tracking Time with Projects                                                                                                        | Teaches project users about the functions of timesheets and Intelligent time in Sage Intacct Projects.                                                                                                         |
| Processing Expenses with Projects                                     | Teaches project users about the functions of expenses in Sage Intacct Projects.                                                    |
| Managing Labor Costing with Projects                                  | Teaches project users about managing labor costing in Sage Intacct Projects.                                                       |
| Billing and Invoicing with Projects                                   | Teaches project users about billing and invoicing in Sage Intacct Projects.                                                        |
| Reporting                                                             | Getting Started with Interactive Custom Report Writer                                                                              | Provides an introduction to building reports using the Interactive Custom Report Writer (ICRW). You will learn about reporting areas, how to navigate ICRW, and how to create a new interactive custom report. |
| Sage Intacct Planning                                                 | Navigate Sage Intacct Planning                                                                                                     | Provides an overview of the key features and functionalities available in Sage Intacct Planning.                                                                                                               |
| Create a budget with Sage Intacct data                                | Offers an interactive introduction to building a budget with Sage Intacct data.                                                    |
| The budget tree structure                                             | Introduces you to the budget tree hierarchy within Sage Intacct Planning.                                                          |
| Budget line types                                                     | Offers an interactive exploration of the essential budget line types in Sage Intacct Planning.                                     |
| Manage user permissions                                               | Provides an interactive session on user permissions within Sage Intacct Planning.                                                  |
